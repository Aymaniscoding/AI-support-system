# My Country Mobile AI Assistant

An AI-powered customer support assistant built using n8n, Hugging Face, JavaScript, and Vercel. The application allows users to interact with an AI assistant, generate AI-powered responses, request human support, and automate backend workflows through n8n.

## 🚀 Live Demo

https://mycountrymobileai.vercel.app/work.html

## 📌 Features

- AI-powered customer support chatbot
- Human support escalation via email
- Workflow automation using n8n
- Hugging Face AI model integration
- Real-time frontend and backend communication
- Webhook-based architecture
- Responsive user interface
- Cloud deployment with Vercel

## 🛠️ Tech Stack

## 🏗️ Workflow Architecture

text User  │  ▼ Frontend (Vercel)  │  ▼ n8n Webhook  │  ▼ Switch Node  ├── AI Query  │     │  │     ▼  │  Hugging Face AI  │     │  │     ▼  │  Response to User  │  └── Human Support        │        ▼      Gmail        │        ▼   Support Team Notification
